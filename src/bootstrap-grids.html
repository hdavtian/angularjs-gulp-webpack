Bootstrap grids

.row > col-*-*

This is possibly the most basic usage of the bootstrap grid. A div with class row wrapping other divs with col-*-* classes.

col-sm-4
col-sm-4
col-sm-4
<section class="row">
	<div class="col-sm-4">col-sm-4</div>
	<div class="col-sm-4">col-sm-4</div>
	<div class="col-sm-4">col-sm-4</div>
</section>

.container > .row > col-*-*

When you wrap a basic row with cols with a .container, you center your markup. When you wrap your content with .container-fluid it's full width just like if you didn't have container at all.

.container
col 1
col 2
col 3
.container-fluid
col-sm-4
col-sm-4
col-sm-4
<section class="container">
    <div class="row">
        <div class="col-sm-4 test-box">col 1</div>
        <div class="col-sm-4 test-box">col 2</div>
        <div class="col-sm-4 test-box">col 3</div>
    </section>
</div>

<section class="container-fluid">
	<div class="row">
		<div class="col-sm-4">col-sm-4</div>
		<div class="col-sm-4">col-sm-4</div>
		<div class="col-sm-4">col-sm-4</div>
	</div>
</section>

Centering bootstrap columns using offsets

Here's one method to center columns inside a row, by using offset. This method will not require any additional changes in markup or css. The formula is: "set an offset equal to half of the remaining size of the row"

col-xs-10 col-xs-offset-1

	</div>
</div>
